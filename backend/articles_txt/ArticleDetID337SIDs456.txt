OU.edu Custom URL Requests
A URL Request allows OU website owners to request custom URLs for their domain, redirects to another OU web address or to set up addresses for departmental file shares. These new domain requests will be approved for content and clarity before being enabled on the ou.edu domain. Your site should also comply with all
OU web policies
. If you are a OU web site owner or developer and wish to request a custom URL, please
visit the Service Catalog
.
